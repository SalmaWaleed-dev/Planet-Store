# 🌿 Planet Shop

A modern and elegant web project developed as part of our studies at the Egyptian E-Learning University (EELU).
Greenora is an online store that offers a variety of indoor plants and decorative pots, designed to bring nature closer to your space.

---

## ✨ About the Project

This project focuses on building a clean and user-friendly e-commerce interface for plant lovers 🌱
It aims to apply front-end development skills while creating a visually appealing and responsive design.

---

## 🚀 Features

* 🌱 Browse different types of indoor plants
* 🪴 Explore a collection of stylish pots
* 📱 Responsive design for all devices
* 🎨 Clean and modern UI
* 🛒 Simple and intuitive shopping experience

---

* HTML5
* CSS3

* PHP
